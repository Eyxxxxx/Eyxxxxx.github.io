Hexo-问题1-hexo deploy 时每次要输入账号密码

发表于 2021-01-25 更新于 2021-03-25 分类于 Hexo ， Q&A

环境说明

OS: Win10
Hexo theme： NexT
NexT: v7.8.0   (FYI: NexT 最新版为v8.0.0)

注意事项

以下未特别声明时皆修改NexT主题的主题配置文件_config.yml文件. 在_config.yml中搜索关键词完成相关任务.
注意区别主题配置文件与站点配置文件.

hexo deploy 时每次要输入账号密码

repository: 应填写git@github.com:username/username.github.io.git. (以保证使用SSH协议)

例如:

deploy:
 type: git
 repository: git@github.com:Eyxxxxx/Eyxxxxx.github.io.git
 branch: master

注意

  • repository 不应写为https://github.com/Eyxxxxx/Eyxxxxx.github.io.git ←这是用https协议的, 会需要每次输入账号密码.
  • 每一项的英文冒号后面皆有空格, 不应错删或者使用错误的符号.
  • 若此处deploy更改完仍有问题, 注意检查是否成功部署GitHub的SSH配置 参考链接 - Git SSH Key 生成步骤
  • 其他参考 - Git 工具 - 凭证存储
